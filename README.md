# Calculator App (Java Swing)

A modern calculator application built using Java Swing that provides a clean graphical interface and supports basic arithmetic operations along with utility functions.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 📊 Percentage Calculation
- 🔄 Positive/Negative Toggle (`+/-`)
- 🧹 All Clear (`AC`)
- 🔢 Decimal Number Support
- 🎨 Custom iOS-inspired User Interface
- 🖥️ Grid-based Button Layout
- 🧮 Automatic Removal of Unnecessary `.0` in Results

## 🛠 Technologies Used

- Java
- Swing
- AWT
- BorderLayout
- GridLayout
- ActionListener

## 📂 Project Structure

```text
Calculator.java    # Main calculator implementation
App.java           # Application entry point
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/calculator.git
cd calculator
```

### Compile the Program

```bash
javac *.java
```

### Run the Application

```bash
java App
```
